## Project Overview

**Orion** is a trading signals platform implementing the Option for Income (OFI) strategy:
- Screens S&P 500 stocks for bullish conditions
- Detects oversold bounces with volume confirmation
- Finds ATM put options for income generation
- Sends email alerts for opportunities

## Loop Instructions

Use `PROMPT_plan.md` for planning mode:
- Analyzes gaps between specs and code
- Generates prioritized task list
- Does NOT implement anything

Use `PROMPT_build.md` for building mode:
- Implements tasks from plan
- Runs tests after each change
- Commits when tests pass
- Updates plan with findings
